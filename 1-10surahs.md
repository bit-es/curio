| **2. SURAH AL-HUMAZAH** | | | |
| Verse 1 | وَيْلٌ لِّكُلِّ | Wailul-likulli | Woe to every |
| | هُمَزَةٍ لُّمَزَةٍ | humazatil-lumazah | scorner and mocker, |
| Verse 2 | الَّذِي جَمَعَ مَالًا | Alladzii jama'a maalan | who collects wealth |
| | وَعَدَّدَهُ | wa 'addadah | and continuously counts it. |
| Verse 3 | يَحْسَبُ أَنَّ مَالَهُ | Yahsabu anna maalahuu | He thinks that his wealth |
| | أَخْلَدَهُ | akhladah | will make him immortal. |
| Verse 4 | كَلَّا لَيُنبَذَنَّ | Kallaa layumbadzanna | No! He will surely be thrown |
| | فِي الْحُطَمَةِ | fil-huthamah | into the Crusher. |
| Verse 5 | وَمَا أَدْرَاكَ مَا الْحُطَمَةُ | Wa maa adraaka mal-huthamah | And what can make you know what is the Crusher? |
| Verse 6 | نَارُ اللَّهِ الْمُوقَدَةُ | Naarul-laahil-muuqadah | It is the ignited Fire of Allah, |
| Verse 7 | الَّتِي تَطَّلِعُ عَلَى الْأَفْئِدَةِ | Allatii tattholi'u 'alal-af'idah | which mounts up to the hearts. |
| Verse 8 | إِنَّهَا عَلَيْهِم مُّصْدَةٌ | Innahaa 'alaihim mush-shadah | Indeed, it will be closed over them |
| Verse 9 | فِي عَمَدٍ مُّمَدَّدَةٍ | Fii 'amadim-mumaddadah | in extended columns. |

| **3. SURAH AL-FIL** | | | |
| Verse 1 | أَلَمْ تَرَ كَيْفَ فَعَلَ | Alam tara kaifa fa'ala | Have you not seen how dealt |
| | رَبُّكَ بِأَصْحَابِ الْفِيلِ | rabbuka bi ash-haabil-fiil | your Lord with the companions of the elephant? |
| Verse 2 | أَلَمْ يَجْعَلْ كَيْدَهُمْ | Alam yaj'al kaidahum | Did He not make their plot |
| | فِي تَضْلِيلٍ | fii tadhliil | go astray? |
| Verse 3 | وَأَرْسَلَ عَلَيْهِمْ | Wa arsala 'alaihim | And He sent against them |
| | طَيْرًا أَبَابِيلَ | thairan abaabiil | birds in flocks, |
| Verse 4 | تَرْمِيهِم بِحِجَارَةٍ | Tarmiihim bihijaa-ratim | striking them with stones |
| | مِّن سِجِّيلٍ | min sijjiil | of hard clay, |
| Verse 5 | فَجَعَلَهُمْ | Faja'alahum | so He made them |
| | كَعَصْفٍ مَّأْكُولٍ | ka'ashfim-ma'kuul | like chewed-up straw. |

| **4. SURAH QURAISH** | | | |

| **5. SURAH AL-MA'UN** | | | |
| Verse 1 | أَرَأَيْتَ الَّذِي | Ara'aital-ladzii | Have you seen the one who |
| | يُكَذِّبُ بِالدِّينِ | yukadz-dzibu bid-diin | denies the Recompense? |
| Verse 2 | فَذَٰلِكَ الَّذِي | Fadzaalikal-ladzii | For that is the one who |
| | يَدُعُّ الْيَتِيمَ | yadu''ul-yatiim | repels the orphan, |
| Verse 3 | وَلَا يَحُضُّ | Wa laa yahudh-dhu | and does not encourage |
| | عَلَىٰ طَعَامِ الْمِسْكِينِ | 'alaa tho'aamil-miskiin | the feeding of the poor. |
| Verse 4 | فَوَيْلٌ لِّلْمُصَلِّينَ | Fawailul-lil musholliin | So woe to those who pray, |
| Verse 5 | الَّذِينَ هُمْ عَن صَلَاتِهِمْ | Alladzii nahum 'an sholaatihim | who are concerning their prayer |
| | سَاهُونَ | saahuun | heedless, |
| Verse 6 | الَّذِينَ هُمْ يُرَاءُونَ | Alladziina hum yuraa'uun | those who make show [of their deeds] |
| Verse 7 | وَيَمْنَعُونَ الْمَاعُونَ | Wa yamna'uunal-maa'uun | and withhold simple assistance. |
